# V1.1-02 — Module Documentaire : Roadmap tRPC

## Objectif
Implémenter la génération et gestion des documents PDF (parcelles, dossiers) avec vérification publique et audit trail.

## Procédures tRPC à Ajouter

### Admin Router

#### `admin.generateParcelPdf`
- **Input** : `{ parcelId: number }`
- **Output** : `{ url: string, fileKey: string, documentId: number }`
- **Description** : Génère un PDF de la parcelle (informations publiques + timeline)
- **Audit** : `admin.generateParcelPdf`
- **Sécurité** : adminProcedure

#### `admin.generateDossierPdf`
- **Input** : `{ dossierId: number }` (future — pour crédit)
- **Output** : `{ url: string, fileKey: string, documentId: number }`
- **Description** : Génère un PDF du dossier crédit
- **Audit** : `admin.generateDossierPdf`
- **Sécurité** : adminProcedure

#### `admin.listGeneratedDocuments`
- **Input** : `{ parcelId?: number, limit?: number, offset?: number }`
- **Output** : `GeneratedDocument[]`
- **Description** : Liste les documents générés (paginé)
- **Audit** : `admin.listGeneratedDocuments`
- **Sécurité** : adminProcedure

#### `admin.getGeneratedDocumentDownloadUrl`
- **Input** : `{ documentId: number }`
- **Output** : `{ url: string, expiresAt: Date }`
- **Description** : Obtient une URL de téléchargement temporaire (presigned S3)
- **Audit** : `admin.downloadGeneratedDocument`
- **Sécurité** : adminProcedure

### Verify Router

#### `verify.check` — Enrichissement
- **Modification** : Ajouter métadonnées document si `tokenType === 'document'`
- **Output enrichi** : 
  ```typescript
  {
    valid: boolean,
    tokenType: string,
    status: string,
    documentMetadata?: {
      documentType: string,
      generatedAt: Date,
      parcelReference: string,
      // Pas de PII
    }
  }
  ```
- **Audit** : Inchangé (`verify.check`)
- **Sécurité** : publicProcedure + rate limit

### Citizen Router (Future)

#### `citizen.downloadDocument`
- **Input** : `{ documentId: number }`
- **Output** : `{ url: string, expiresAt: Date }`
- **Description** : Citoyen télécharge son document (owner-only)
- **Audit** : `citizen.downloadDocument`
- **Sécurité** : protectedProcedure + isolation ownerId

## Tables à Ajouter

### `generated_documents`
```sql
CREATE TABLE generated_documents (
  id INT AUTO_INCREMENT PRIMARY KEY,
  parcelId INT NOT NULL,
  documentType ENUM('parcel_pdf', 'dossier_pdf', 'attestation') NOT NULL,
  fileUrl VARCHAR(512),
  fileKey VARCHAR(256),
  status ENUM('draft', 'published', 'archived') DEFAULT 'published',
  metadata JSON,
  createdById INT,
  createdAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updatedAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  FOREIGN KEY (parcelId) REFERENCES parcels(id),
  FOREIGN KEY (createdById) REFERENCES users(id)
);
```

### `verify_tokens` — Enrichissement
```sql
ALTER TABLE verify_tokens
ADD COLUMN tokenType ENUM(..., 'document') 
-- Ajouter 'document' aux tokenType existants
```
