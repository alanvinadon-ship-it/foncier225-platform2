# Dossier d'Interconnexion API — Foncier225

## Vue d'ensemble

Ce dossier contient la documentation complète pour l'interconnexion de la plateforme Foncier225 avec les trois infrastructures numériques majeures de l'État ivoirien en matière de gestion foncière.

## Systèmes concernés

| Système | Organisme | Périmètre |
|---------|-----------|-----------|
| **SIGFU** | MCLU | Foncier urbain — 49 procédures, suivi des demandes d'actes |
| **IDUFCI** | MCLU / DMISSA | Identifiant unique parcelle (20 caractères) — attribution, vérification, morcellement |
| **SIFOR-CI** | AFOR | Foncier rural — certificats fonciers, délimitations villageoises (8 modules) |

---

## Procédure de soumission auprès de l'ARTCI

1. Compléter les champs `[À compléter]` dans le cahier des charges avec les informations de la société.
2. Rassembler les 14 pièces jointes listées à la section 11 du cahier des charges.
3. Déposer le dossier physique (original + 2 copies) au secrétariat de l'ARTCI.
4. Envoyer simultanément la version numérique à : **ConsultationpubliqueAPI@artci.ci**
5. Délai d'instruction : 60 jours ouvrables.
